WHAT THIS MILESTONE MEANS

Every combat and command MECHANIC the game will ship is now built, tuned,
and trace-proven. From here forward the project builds INTERFACE and CONTENT
on a frozen simulation. There is no combat or command mechanic left to invent.

The simulation is deterministic (seeded), sim/render thread-separated, and
runs the 40-block reference battle at 60 FPS on designer hardware.

=====================================================================
THE COMPLETE SIMULATION (WO-001 .. WO-033)

Core model (Phase 1):
- Three pillars: Strength (bodies), Cohesion (morale/will), Push (the shove).
  Battles are won by breaking will - units rout ~67% strength remaining.
- Edge-based contact: front/left/right/rear classification (32-case compass),
  per-channel morale multipliers, corner blending. Flanks kill by terror.
- Hybrid rout/RALLY, neighbor-rout shock cascade, blob-proof frontage caps,
  R19 pinning.

Combat systems (Phase 2):
- Real damage: 9 stats, armor-class x damage-type matrix, anti-armor, 20%
  chip floor, time-domain-correct scale constants (melee/ranged separate).
- Ranged: volleys, falloff, ammo tiers, 3 fire doctrines, 15m dead-zone
  panic, 70% friendly fire.
- Mounted: momentum charge (Impact = mass x REAL closing velocity x Str%),
  physical charge gait/gallop (no hidden constants), mass/inertia via Profile,
  directional charge shock, three-tier brace (instinctive/set/unaware).
- Magnetism: surface-gap gravity, live Agility (rotation/disengage), fighting-
  withdrawal, rotation-under-contact drain (phalanx wheels cost real cohesion).
- Slope: general height field; push/range/speed modifiers emerge; charges gain
  downhill velocity with ZERO slope-specific charge code (R6 vindicated twice).
- Quality of the Day (R21): persistent per-battle variance sigma=0.045, tuned
  to the designer's curve (0%->50, 3%->63, 5%->73, 10%->86, 20%->98). Maneuver
  advantages remain decisive - no good day saves a unit from a rear charge.

Command layer (Phase 3, sim side):
- Order system: 3-step queues, primitives (advance/hold/absolute_hold/attack/
  feign/flank/swing_and_charge), triggers (proximity/engage/rout/cohesion/
  timer/order_started/horn). Executes Cannae from a data file (S42, 11/11).
- Concealment & ambush (Sec 10): forest/shrub detection, Profile multipliers,
  fit rule, permanent reveal, brace-clock-from-reveal. Ambush worth a measured
  53 cohesion points vs a visible flank (Teutoburg S45).
- Feigned retreat: fighting-withdrawal + a 2s deception window shown to the
  enemy without contaminating enemy logic. Baits the pursuer (S52, 11/11);
  low-skill feints self-rout (S53).
